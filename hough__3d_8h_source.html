131  /** \brief Class implementing a 3D correspondence grouping algorithm that can deal with multiple instances of a model template
132  * found into a given scene. Each correspondence casts a vote for a reference point in a 3D Hough Space.
133  * The remaining 3 DOF are taken into account by associating each correspondence with a local Reference Frame.
134  * The suggested PointModelRfT is pcl::ReferenceFrame
135  *
136  * \note If you use this code in any academic work, please cite the original paper:
137  * - F. Tombari, L. Di Stefano:
138  * Object recognition in 3D scenes with occlusions and clutter by Hough voting.
139  * 2010, Fourth Pacific-Rim Symposium on Image and Video Technology
140  *
141  * \author Federico Tombari (original), Tommaso Cavallari (PCL port)
142  * \ingroup recognition
143  */
