Detailed Description

template<class MeshT>
class pcl::geometry::VertexAroundVertexCirculator< MeshT >

Circulates counter-clockwise around a vertex and returns an index to the terminating vertex of the outgoing half-edge (the target).

The best way to declare the circulator is to use the method pcl::geometry::MeshBase::getVertexAroundVertexCirculator ().

Template Parameters
MeshTMesh to which this circulator belongs to.
Note
The circulator can't be used to change the connectivity in the mesh (only const circulators are valid).

Martin Saelzle
